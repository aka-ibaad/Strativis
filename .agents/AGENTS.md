# Workspace Rules

## ANIMATION RULES
- **Motion System**: Prefer using a motion system like Framer Motion for rich, declarative UI animations.
- **Fade in on Scroll**: Apply smooth fade-in animations as elements enter the viewport when scrolling.
- **Stagger Animations**: Apply staggered delays to child lists/grids to ensure sequential entering effects.
- **Hover Lift Effects**: Apply responsive micro-lifts, shifts, and scaling transitions to clickable interactive elements on hover.
- **Page Transitions**: Use fluid layout animations, slide/fade page transitions, and smooth route change visuals.
- **Smooth Micro-interactions**: Ensure buttons, dropdowns, inputs, and toggles have seamless transition-states and organic easing.

## DESIGN SYSTEM RULES
- **Spacing Rules**: Use a strict, proportional spacing scale (e.g., 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px) for layout alignment.
- **Typography Hierarchy**: Establish clear type scales (e.g., display, headers, body, caption) with precise line heights and font weights (e.g., Outfit, Inter).
- **SaaS Layout Structure**: Build standard, responsive layouts featuring sticky headers, collapsible sidebars, card-based dashboards, and content-first grids.
- **Color Psychology**: Use premium, curated, harmonious palettes (tailored dark modes, high-contrast states, and subtle gradients) rather than generic fallback colors.
- **Conversion Optimization**: Strategically position calls-to-action (CTAs), prioritize visual hierarchy, design clear hero banners, and ensure minimal-friction input flows.

## UI BLOCK SOURCE
- **Component Library Reference**: Leverage design ideas and structural layouts matching high-quality communities such as [21st.dev](https://21st.dev/community/components).
- **Hero Sections**: Create high-impact, visual headers with engaging headlines, CTAs, and ambient lighting/visual backgrounds.
- **Pricing Cards**: Structure clean pricing tiers highlighting features, value propositions, and primary/secondary action buttons.
- **Feature Grids**: Lay out products or key benefits using interactive grids, hover-active cards, or bento layouts.
- **Navbars**: Maintain clean, responsive header navigation with hover-states, dropdown menus, and utility buttons.
- **Dashboards**: Organize dashboards with analytical cards, graphs, clean sidebar navigation, and user control hubs.
